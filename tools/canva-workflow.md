**One-time setup saves hours later:**

1. **Create new design:**

   - Duration: 24 seconds

2. **Set up 3-page template:**

   - Page 3: Scene 3 (16-24s)

3. **Add placeholder elements:**
   - Background layers
   - Text overlay positions
   - Character video position

## ðŸŽ¬ Production Workflow (Per Video)

### Step 1: Import Lip-Sync Video

1. **Open master template**
2. **Upload** Hedra lip-sync video
3. **Drag** to canvas
4. **Remove background** (if needed):
   - Select video
   - Click "Edit video"
   - Choose "Background remover"
   - Wait for processing

---

### Step 2: Build Scene 1 (0-8s Setup)

**Follow scene plan from `/production/video-plans/`**

1. **Add background:**
   - Drag background image to page 1
   - Resize to fill frame
   - Send to back layer

2. **Position character video:**
   - Place lip-sync video on top
   - Resize/position per scene plan
   - Adjust timing: 0-8 seconds

3. **Add text overlay** (if specified):
   - Click "Text"
   - Type character name/intro text
   - Style: Bold, white text, black outline
   - Position: Top or bottom (per scene plan)
   - Animate: Fade in at specified time

4. **Apply effects** (if specified):
   - Lighting adjustments
   - Color filters
   - Shadows/glows

---

### Step 3: Build Scene 2 (8-16s Build)

**Switch to page 2:**

1. **Change background** (different from scene 1)
2. **Adjust character position/size** (create visual variety)
3. **Update text overlay** (if specified)
4. **Adjust timing:** 8-16 seconds
5. **Add transition** from scene 1:
   - Quick cut (no fade)
   - Or subtle dissolve

---

### Step 4: Build Scene 3 (16-24s Punchline)

**Switch to page 3:**

1. **Final background** (payoff shot)
2. **Character final position**
3. **Punchline text overlay:**
   - Appears at specified time
   - Larger/bolder for emphasis
   - May include emoji
4. **Timing:** 16-24 seconds
5. **Add transition** from scene 2

---

### Step 5: Final Polish

1. **Review full video:**
   - Play from start to finish
   - Check timing (exactly 24 seconds)
   - Verify text is readable
   - Ensure smooth transitions

2. **Adjust audio** (if needed):
   - Volume levels
   - Fade in/out

3. **Add branding** (optional):
   - Small logo/watermark
   - Consistent placement across all videos

---

### Step 6: Export

1. **Click "Share"**
2. **Select "Download"**
3. **Settings:**
   - File type: MP4 Video
   - Quality: High (1080p)
   - Pages: All pages
4. **Download**
5. **Rename file:**
   - Format: `[character-id]_[song-id]_FINAL.mp4`
   - Example: `grumpy-sloth_creep-radiohead_FINAL.mp4`
6. **Save to:** `assets/videos/final/`

## ðŸŽ¨ Design Best Practices

### Text Overlays

**Readability:**
- Font size: 60-80pt minimum
- Bold fonts (Impact, Bebas Neue, Montserrat Bold)
- White text with black outline (always readable)
- Avoid busy backgrounds behind text

**Positioning:**
- Top third or bottom third (never center)
- Leave breathing room around edges
- Consistent placement across videos

### Backgrounds

**Visual Variety:**
- Change background for each scene
- Use different lighting/colors
- Create depth (foreground/background elements)

**Don't Overdo:**
- Backgrounds support, don't distract
- Character should be focal point
- Avoid busy/cluttered backgrounds

### Transitions

**Keep Simple:**
- Quick cuts work best for comedy
- Subtle dissolves if needed
- No fancy wipes/spins (distracting)

## ðŸš¨ Common Issues & Solutions

### Issue: Background removal looks bad

**Solutions:**
- Use higher quality source video
- Manually refine edges in Canva
- Use solid background instead of removing
- Try different Hedra export settings

### Issue: Text is hard to read

**Solutions:**
- Increase font size
- Add thicker outline
- Use simpler background
- Change text color
- Add semi-transparent box behind text

### Issue: Video is too long/short

**Solutions:**
- Trim individual scenes in Canva
- Adjust timing of each page
- Re-export from Hedra with correct duration

### Issue: Transitions look choppy

**Solutions:**
- Use quick cuts instead of transitions
- Ensure scenes are exactly 8 seconds each
- Check frame rate consistency
